Acerca de
Desarrollado por Rodolfo Lara en GPT-5 mini de OpenAI y un servicio serverless en Azure para procesar respuestas.

Privacidad y uso de datos
Este chat guarda temporalmente tus mensajes en tu navegador y envía solo los últimos turnos a una función en la nube para generar la respuesta mediante la API de OpenAI. Al recargar esta página o cerrar la ventana, cualquier dato se borra de tu navegador.

No accedemos a tus conversaciones. No guardamos tus datos, y el procesamiento se realiza de forma anónima.

OpenAI procesa las solicitudes únicamente para generar la respuesta y, según su propia política, puede retenerlas por un tiempo limitado con fines de seguridad. No compartimos tu información con otros terceros.
